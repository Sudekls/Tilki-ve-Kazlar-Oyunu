Bu proje, Algoritmik Oyun Kuramı dersi kapsamında geliştirilmiştir. Geliştirme sürecinde Visual Studio Code (VS Code) kullanılmıştır.

Projede, klasik Tilki ve Kazlar oyunu temel alınarak farklı zorluk seviyelerine sahip bir oyun tasarlanmıştır. Oyuncular, oyun başlamadan önce istedikleri zorluk seviyesini seçebilir ve Tilki veya Kaz tarafında oynama seçeneklerinden birini tercih edebilirler. Bu sayede kullanıcılar farklı stratejiler deneyimleyerek oyunu çeşitli senaryolarda oynama imkânı bulmaktadır.
